# sudoku solver
my attempt at solving sudokus with python :>

for now it can solve (*actually outputs multiple solutions if indeed they exist) a grid given in main, might edit it later and make it more interactive, idk.
feel free to pull request.

also i didnt add a gui but i did pretty-print it, so it looks kinda nice imo.
